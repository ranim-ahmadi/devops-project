# DevOps Project - Jenkins Pipeline

This project implements a **CI/CD pipeline** using Jenkins with a Shared Library.  

## Features

- Checkout code from Git  
- Maven build and unit tests  
- Generate JaCoCo code coverage reports  
- Run SonarQube analysis  
- Deploy artifacts to Nexus  
- Build and push Docker images  
- Deploy application with Docker Compose  
- Scan filesystem for vulnerabilities with Trivy
